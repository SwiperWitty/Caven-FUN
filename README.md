# Caven-FUN

基于Cavendish 3.14做的 [Cavendish 3.14 软件源](https://github.com/SwiperWitty/Core-Caven32)    [Cavendish 3.14 硬件（立创EDA）](https://oshwhub.com/SWP_MCU/aeefd84f983743a1af339ab6bbbd6a80)



详情：

- MCU-AT32F415RCT6
- 屏幕-ST7789
- 语音模块
- 电池充放-IP5306

|  网络ID  |    GPIO    |     功能     |
| :------: | :--------: | :----------: |
|   LED    |    PA12    |      灯      |
|  EN_SKP  |    PA11    | 启动语音模块 |
| USER_IO3 |    PD2     |     方向     |
|  串口2   |  PA2、PA3  |  语音<->MCU  |
|  串口3   | PC10、PC11 |   语音下载   |
|   ADC1   |    PA4     |   电池电压   |
|   ADC2   |    PA5     |   光线采集   |
